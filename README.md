# Frontend Mentor - Solução de cartão de perfil

Esta é uma solução para o [desafio do componente do cartão de perfil no Frontend Mentor](https://www.frontendmentor.io/challenges/profile-card-component-cfArpWshJ).

### O desafio

- Construir o projeto de acordo com os designs fornecidos e ajuda dos meus professores

## Meu processo

### Construído com

- Marcação HTML5 semântica
- Propriedades personalizadas CSS
- flexbox
- Fluxo de trabalho voltado para dispositivos móveis
- PerfectPixel
- Responsive viewer

### O que eu aprendi

Durante o curso do desafio eu aprende  a como funciona o uso do container  e como deixar o projeto responsivo

```html
<div class="info">
   <p class="price"> 0.041 ETH</p>
   <p class="days-left">3 days left</p>
</div>

```

```css
.nft-card .info .days-left::before{
    content: '';
    background-image: url(../images/icon-clock.svg);
    background-repeat: no-repeat;
    background-position: center;
    width: 15px;
    display: inline-block;
    margin-right: 5px;

```

### Recursos úteis

- PerfectPixel- Isso me ajudou a começar o meu desafio, como não tenho o premium ele me ajudou a ter um norte, com a medidas e tamanhos do projeto. -Gostei muito desse padrão e vou usá-lo daqui para frente.
- Responsive viewer - como ele eu consegui visualizar meu projetos em todos os tamanhos renponsivos
